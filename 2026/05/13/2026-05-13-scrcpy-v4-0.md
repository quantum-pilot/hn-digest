# Scrcpy v4.0

- Score: 350 | [HN](https://news.ycombinator.com/item?id=48114356) | Link: https://github.com/Genymobile/scrcpy/releases/tag/v4.0

- TL;DR  
  Scrcpy 4.0 is a significant upgrade to the open‑source Android mirroring/control tool. It migrates to SDL3 (better maintenance, aspect‑ratio‑locked resizing), adds flex virtual displays that resize with the desktop window, dynamic camera torch/zoom controls, “keep‑active” without changing system settings, customizable backgrounds, and clearer disconnect UX. Bugs like Meta Quest flicker and high CPU on silent audio are fixed. HN users describe scrcpy as “mind‑blowing” and indispensable for tethering hacks, dead‑screen recovery, and even bypassing screenshot restrictions.

- Comment pulse  
  - Ad‑hoc networking and tethering → scrcpy lets users enable USB sharing and manage phones hung in windows for neighbor Wi‑Fi, complementing other improvised “dongle” hacks.  
  - Everyday power tool → commenters call scrcpy mind‑blowing for Android control, flex displays, DeX on unsupported phones, and capturing content that blocks screenshots or recordings.  
  - Data‑recovery lifeline → saves data and 2FA from dead‑screen phones if ADB trust was pre‑set—counterpoint: some fear Google could eventually clamp down.
