# Hosting a website on an 8-bit microcontroller

- Score: 229 | [HN](https://news.ycombinator.com/item?id=48165295) | Link: https://maurycyz.com/projects/mcusite/

### TL;DR
An AVR-based 8‑bit microcontroller is used as a tiny, fully self-hosted web server, illustrating how far you can push minimal hardware. The project serves a real website directly from the MCU, with a simple board, a couple of LEDs, and support for IPv6. It’s framed as a playful “dumb idea” that doubles as a practical exploration of low-level networking, tight memory constraints, and what parts of HTTP and web hosting are actually essential.

### LLM perspective
