# Gemini API File Search is now multimodal

- Score: 145 | [HN](https://news.ycombinator.com/item?id=48080702) | Link: https://blog.google/innovation-and-ai/technology/developers-tools/expanded-gemini-api-file-search-multimodal-rag/

### TL;DR
Google’s Gemini API File Search now does multimodal retrieval: it indexes and searches text and images together using Gemini Embedding 2, supports custom key‑value metadata filters for more targeted RAG, and returns page-level citations so answers can be traced back to specific PDF pages. Google pitches it as “photographic memory” for agents, with early users reporting accurate, low‑latency mixed‑media search. Hacker News reactions focus less on the tech and more on Google’s poor Gemini UX, privacy worries, and weak product execution versus competitors.
