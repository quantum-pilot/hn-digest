# Space Cadet Pinball on Linux

- Score: 309 | [HN](https://news.ycombinator.com/item?id=48082968) | Link: https://brennan.io/2026/05/09/pinball-and-escrow/

- TL;DR  
The post explains how to run the classic Windows XP game Space Cadet Pinball natively on Linux via a Flatpak port of a reverse‑engineered engine. It walks through installing the Flatpak, then upgrading graphics by swapping in higher‑resolution data files from the related Full Tilt! Pinball release. The author notes subtle gameplay differences between data sets, reflects on the tension between copyright and software preservation, and argues for models like source‑code escrow to eventually free abandoned proprietary software.
