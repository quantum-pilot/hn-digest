# OpenBSD 7.9

- Score: 371 | [HN](https://news.ycombinator.com/item?id=48192882) | Link: https://www.openbsd.org/79.html

### TL;DR

OpenBSD 7.9 (the 60th release, “PinkPuffy”) is a tightening-and-polish release focused on hardware, networking, and security hardening. It adds broader arm64/riscv64/Sophgo/SpacemiT support, better Apple Silicon/Virtualization behavior, and a scheduler that understands mixed-performance cores. Networking sees VLAN-aware virtual bridges, new pf rate/state limiters, IPv6 autoconf enabled by default, and significant Wi-Fi work including basic 802.11ax and many iwx(4) fixes. Security gets further pledge/unveil refinements, stricter kernel interfaces, and LibreSSL 4.3 with a hybrid post-quantum TLS keyshare. HN discussion centers on OpenBSD’s security reputation, hardware gaps vs Linux/FreeBSD, and a shift away from Exim toward OpenSMTPD.

### LLM perspective

- View: This release deepens OpenBSD’s niche as a secure, conservative, general-purpose UNIX, not a feature-chasing desktop OS.  
- Impact: Network operators, homelabbers, and mail/DNS hosts gain more capable hardware targets and safer defaults with modest configuration effort.  
- Watch next: Adoption of PQC TLS modes, further Wi-Fi and ARM/RISC-V support, and how far ports (e.g., OpenSMTPD) spread beyond OpenBSD.
