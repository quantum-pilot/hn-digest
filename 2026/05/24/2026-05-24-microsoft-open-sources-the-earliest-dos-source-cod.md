# Microsoft open-sources “the earliest DOS source code discovered to date”

- Score: 410 | [HN](https://news.ycombinator.com/item?id=48253386) | Link: https://arstechnica.com/gadgets/2026/04/microsoft-open-sources-the-earliest-dos-source-code-discovered-to-date/

### TL;DR
Microsoft has released the earliest known DOS source code: 86-DOS 1.00, early PC‑DOS 1.00 kernel snapshots, and utilities like CHKDSK. The code predates the MS‑DOS name and only survived on paper, so a volunteer “DOS Disassembly Group” laboriously scanned and hand-corrected dot‑matrix listings. The release extends Microsoft’s recent trend of open‑sourcing historic software (earlier MS‑DOS versions, Zork, 3D Movie Maker), giving historians and hobbyists a clearer view into the technical and business roots of the IBM PC era.
