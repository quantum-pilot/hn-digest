# A desktop made for one

- Score: 217 | [HN](https://news.ycombinator.com/item?id=47997947) | Link: https://isene.org/2026/05/Audience-of-One.html

- TL;DR  
  An engineer rebuilt nearly his entire Linux desktop stack—window manager, shell, editor, mail, calendar, file manager—as tiny tools he wrote himself, on an x86_64 assembly base with a Rust TUI layer. Using Claude Code as a coding partner, he compressed what once took years into weeks, discovering most “complexity” serves other users’ needs. He’s not publishing a product; his point is that AI plus modern tooling now make build‑your‑own, audience‑of‑one software practical—and HN readers see a broader movement forming.
