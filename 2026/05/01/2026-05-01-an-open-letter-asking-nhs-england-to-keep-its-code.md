# An open letter asking NHS England to keep its code open

- Score: 217 | [HN](https://news.ycombinator.com/item?id=47975785) | Link: https://keepthingsopen.com

### TL;DR
An open letter urges NHS England to reverse a policy (SDLC‑8) that makes its software repositories private, arguing that publicly funded code should remain open and that transparency strengthens, not weakens, security. The authors cite UK government and NHS standards requiring new source code to be open, liken openness to an “immune system,” and reject security-through-obscurity. HN commenters mostly see the closure as optics driven by AI‑hacking fears, with some calling for a more nuanced balance for safety‑critical systems.
