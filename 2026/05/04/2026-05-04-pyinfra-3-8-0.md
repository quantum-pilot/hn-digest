# PyInfra 3.8.0

- Score: 212 | [HN](https://news.ycombinator.com/item?id=48008083) | Link: https://github.com/pyinfra-dev/pyinfra/releases/tag/v3.8.0

### TL;DR
PyInfra 3.8.0 is a large release of the Python‑native infrastructure automation tool positioned as an alternative to Ansible/Chef/Salt. It tightens security (systematic quoting of untrusted values, command‑precondition guards), broadens platform support (Paramiko 4, macOS/Python 3.13+, CI on Python 3.14), and adds many operations/facts, especially around Docker (build, compose, login/logout, richer facts), git (depth, smarter pull), and GPG keyrings. The project also leans into AI, adding coding‑agent support and a PR‑review “skill.” HN discussion centers on “infrastructure as Python” versus YAML‑based tools.
