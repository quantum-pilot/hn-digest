# Explore Wikipedia Like a Windows XP Desktop

- Score: 479 | [HN](https://news.ycombinator.com/item?id=48146129) | Link: https://explorer.samismith.com/

### TL;DR
A web toy turns Wikipedia and Wikimedia Commons into a faux Windows XP desktop and file explorer. Wikipedia categories become folders, and articles appear as documents you can browse hierarchically. A separate media explorer lets you wander Commons categories and right‑click images to set them as your desktop background. An experimental “Geofile Explorer” treats Earth as a folder tree, where you can upload geotagged images or leave text notes, blending nostalgic UI with exploratory knowledge and map browsing.
