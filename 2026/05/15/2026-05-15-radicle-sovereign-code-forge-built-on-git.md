# Radicle: Sovereign {code forge} built on Git

- Score: 207 | [HN](https://news.ycombinator.com/item?id=48147603) | Link: https://radicle.dev/

### TL;DR
Radicle is a peer‑to‑peer, Git‑based “code forge” that replaces centralized hosts like GitHub with a local‑first, cryptographically signed, gossip‑replicated network. Issues, discussions, and reviews live as Git objects, exposed via CLI, TUI, web, and desktop clients. HN comments debate its permissive licensing versus AGPL, the difficulty of deleting data and leaked secrets in decentralized systems, and its strengths for private repos, experimentation with distributed forges, and emerging “agentic” workflows; some note its recent move off a blocked .xyz domain.
