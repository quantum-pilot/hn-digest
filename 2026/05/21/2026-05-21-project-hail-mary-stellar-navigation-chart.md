# Project Hail Mary – Stellar Navigation Chart

- Score: 487 | [HN](https://news.ycombinator.com/item?id=48225297) | Link: https://valhovey.github.io/gaia-mary/

- TL;DR
    - An interactive Project Hail Mary stellar navigation chart plots Sol, Tau Ceti, and nearby stars in 3D using ESA’s Gaia DR3 catalog for positions and colors. The author pre-renders a skybox from 1.8B stars, then overlays a simplified, non-to-scale solar system for readability. HN discussion explores accuracy versus usability in representing vast distances, how much of the visualization is real astronomy versus fiction, and branches into related space media, games, and learning tools.

- Comment pulse
    - Creator explains using ESA’s Gaia DR3: Python pipeline renders 1.8+ billion stars into custom skybox images, giving realistic positions and approximate colors for the background.
    - Commenters compute large scale errors between orbits and star distances, wanting realism—counterpoint: strict linear scaling ruins UX; logarithmic or mixed scales better support navigation.
    - Thread shares adjacent resources: Sweden’s solar-system model, Elite: Dangerous’s galaxy simulation, explainer videos on time dilation and gravity-well diagrams for understanding space, distance, and orbits.

- LLM perspective
