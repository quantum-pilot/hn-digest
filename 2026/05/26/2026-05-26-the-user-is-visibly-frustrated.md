# The user is visibly frustrated

- Score: 269 | [HN](https://news.ycombinator.com/item?id=48275059) | Link: https://pscanf.com/s/354/

### TL;DR
The author argues coding agents are maddening because their chatty, co-worker-like UX tricks us into expecting human traits—learning, memory, responsibility—that they don’t have. When the agent apologizes, “reflects,” and then repeats the same mistake, it feels like dealing with an incompetent colleague rather than a fallible tool, provoking disproportionate anger. He’d prefer a more robotic, clinical interface or better personal mental habits over anthropomorphizing, but finds it disturbing that tools now require emotional self-defense from their users.
