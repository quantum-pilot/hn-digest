# Open Design: Use Your Coding Agent as a Design Engine

- Score: 166 | [HN](https://news.ycombinator.com/item?id=47985750) | Link: https://github.com/nexu-io/open-design

### TL;DR
Open Design is a local‑first, open‑source alternative to Anthropic’s Claude Design that turns coding agents (Claude Code, Cursor, Copilot, etc.) into a full design engine. It ships “skills,” 70+ brand‑grade design systems (GitHub, Discord, OpenAI, etc.), sandboxed previews, and exports to HTML/PDF/PPTX/MP4. A “craft” layer and linters explicitly fight AI‑slop (default indigo, bad all‑caps tracking, cliché gradients). HN loves the ambition but dislikes the buzzwordy, sales‑deck README and worries about generic, low‑effort AI‑produced design flooding everything.
