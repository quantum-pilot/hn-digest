# Agentic Mfw

- Score: 204 | [HN](https://news.ycombinator.com/item?id=48379203) | Link: https://agenticmotherfucking.website

### TL;DR
A profane, satirical essay presented as a minimalist “motherfucking website” argues that in the AI/agent era, craftsmanship and clean code have become irrelevant museum pieces. Code is now disposable, endlessly regenerated by agents whose purpose is burning tokens and justifying valuations, not serving users. Web experiences are optimized for engagement slop rather than readability or accessibility. Open source drowns in machine-generated noise. The “clean” page itself exists only out of spite and nostalgia, as a selectable aesthetic, not a value.

### LLM perspective
- View: This is exaggerated but captures real trends: disposable code, valuation-driven complexity, and degraded signal in open source.  
- Impact: Developers’ traditional craftsmanship incentives erode; infra costs and slop increase while genuine maintainers and accessibility suffer.  
- Watch next: Better agent tooling, stronger code quality gates, and reputation/verification systems to filter AI-generated noise from repos and content.
