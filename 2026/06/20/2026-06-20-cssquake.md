# CSSQuake

- Score: 442 | [HN](https://news.ycombinator.com/item?id=48608223) | Link: https://cssquake.com/

### TL;DR
CSSQuake is a browser-based Quake remake where the *renderer* is done in CSS (via PolyCSS), while the game logic, physics, and rules are reimplemented in TypeScript. It pulls gameplay data from the original QuakeC `progs.dat` at build time to approximate authentic behavior, though some interactions differ and are being fixed. Performance varies a lot by browser: smooth in Chrome/Firefox but stuttery in WebKit/Safari. It’s a technical art piece rather than an optimized engine, and it still requires JavaScript to run.
