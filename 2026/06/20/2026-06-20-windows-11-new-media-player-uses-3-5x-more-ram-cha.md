# Windows 11 New Media Player Uses 3.5x More RAM, Charges for Popular Video Codecs

- Score: 246 | [HN](https://news.ycombinator.com/item?id=48609343) | Link: https://www.extremetech.com/computing/windows-11s-new-media-player-uses-35x-more-ram-charges-for-popular-video

- TL;DR  
  - Microsoft’s new Windows 11 Media Player replaces both Groove Music and classic WMP but is substantially heavier and less capable. Benchmarks show roughly 3.5× higher idle RAM use and noticeably slower startup for local video. At the same time, Microsoft has removed built‑in support for common codecs, paywalling HEVC playback and dropping native AC‑3 in Windows 11 24H2. Users wanting broad, free format support are steered toward third‑party players like VLC instead.

- LLM perspective  
  - View: Windows stock apps are becoming webby shells, trading efficiency for unified UX and telemetry-friendly architectures.  
  - Impact: Codec unbundling shifts costs to users while reducing Microsoft’s patent exposure; enterprises may standardize on alternative players internally.  
  - Watch next: Track whether regulators question paid codecs on a paid OS, and if OEMs preinstall richer third‑party players to compensate.
