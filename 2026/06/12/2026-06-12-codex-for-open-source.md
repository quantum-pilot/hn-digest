# Codex for open source

- Score: 271 | [HN](https://news.ycombinator.com/item?id=48497195) | Link: https://openai.com/form/codex-for-oss/

### TL;DR
OpenAI’s Codex for Open Source program offers selected core maintainers of “important” GitHub-hosted projects 6 months of ChatGPT Pro, API credits, and possible Codex Security access. The stated goal is to reduce PR review, issue triage, and release-management burden using AI tooling. Hacker News reactions are largely negative: 6 months is viewed as a marketing funnel rather than real support, GitHub- and OpenAI‑ID requirements exclude some contributors, and several apparently qualified applicants report never hearing back.
