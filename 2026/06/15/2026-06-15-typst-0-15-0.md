# Typst 0.15.0

- Score: 263 | [HN](https://news.ycombinator.com/item?id=48544396) | Link: https://typst.app/docs/changelog/0.15.0/

### TL;DR
Typst 0.15.0 is a substantial typesetting upgrade: it adds variable-font support, a new experimental multi-output “bundle” target, MathML-based equation export for HTML, more flexible bibliography handling, and multi-standard PDF conformance. The release also tightens diagnostics, layout, math, and color/printing features while introducing a few breaking changes around paths, layout, and HTML. Hacker News discussion focuses on Typst’s real-world use in book production, integration into broader toolchains, and how well today’s LLMs support the language.

---

### Comment pulse
- Humanities-style workflows still feel fragile → discursive notes with citations can misbehave, including references landing on earlier pages, limiting Typst’s suitability for Chicago-style work.
- Some users run full book pipelines → Word → Pandoc → Typst for print/digital PDFs → back through Pandoc to EPUB, with only minor glue code for edge features.
- Experiences with LLM tooling diverge → some see weak support and frequent hallucinated constructs; others report excellent results when pairing modern models with Typst-aware skills and docs.

### LLM perspective
