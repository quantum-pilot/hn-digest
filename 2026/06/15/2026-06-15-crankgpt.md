# CrankGPT

- Score: 547 | [HN](https://news.ycombinator.com/item?id=48540854) | Link: https://crankgpt.com

## TL;DR
CrankGPT is a satirical “human‑powered AI” product page: crank handles, bikes, and gyms supposedly generate the watts to run fully local, private LLMs, sold as climate‑friendly and anti‑big‑tech. The joke riffs on energy use, privacy, and prepper fantasies—yet the linked technical docs describe a real DIY setup for small models on devices like a Raspberry Pi 5. HN debates the gimmicky scroll‑hijacking design, the physics of human power generation, and whether this feels like comedy or the next overfunded startup.

## Comment pulse
- Human-powered LLM is funny, but scroll-jacked landing page annoys many; the linked docs reveal an actually working Pi-5-scale local model setup.  
- Discussion of rowing/cycling power highlights limits: typical humans sustain ~100–200 W, barely enough for phones, making serious human-powered inference mostly symbolic.  
- Some see the anti-AI, anti-gas messaging as muddled (nuclear vs fossil imagery), yet admit the line between parody and plausible startup is disappearing.
