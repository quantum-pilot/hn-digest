# Iroh 1.0

- Score: 887 | [HN](https://news.ycombinator.com/item?id=48542480) | Link: https://www.iroh.computer/blog/v1

### TL;DR
Iroh 1.0 is a QUIC-based networking stack that replaces dialing IPs with dialing cryptographic keys, giving each app instance a stable, encrypted identity reachable across NATs and changing networks. The 1.0 release locks in wire/API stability, adds official bindings for Python, Node.js, Kotlin, and Swift, and supports custom transports like Tor or BLE behind a single abstraction. Hacker News discussion frames it as “Tailscale or WebRTC for app developers,” debating clarity of messaging, centralization, and what problems it uniquely solves.

### Comment pulse
- Mental model: Tailscale at the application layer → ship connectivity inside your app without separate accounts or external VPN dependency — counterpoint: messaging doesn’t make this clear enough.

- Problem focus: seamless, direct connections across NATs and mobile networks → some see this as a missing “session layer,” others say “IP/DNS already work” and question differentiation.
