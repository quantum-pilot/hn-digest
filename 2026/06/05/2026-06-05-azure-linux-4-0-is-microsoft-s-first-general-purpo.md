# Azure Linux 4.0 is Microsoft's first general-purpose Linux

- Score: 192 | [HN](https://news.ycombinator.com/item?id=48407499) | Link: https://www.boxofcables.dev/azure-linux-4-0-is-microsofts-first-general-purpose-linux/

## TL;DR
Azure Linux 4.0 turns Microsoft’s previously behind-the-scenes CBL-Mariner into a full general-purpose Linux distribution you can explicitly choose on Azure VMs, containers, and soon WSL. It’s now derived from Fedora (currently a Fedora 43 snapshot) with a transparent overlay system documenting every divergence, uses standard dnf5, and ships a modern, hardened stack (kernel 6.18, SELinux, SBOMs, FIPS 140-3 in progress). The distro is intentionally minimal, cloud/server-focused, and marks Microsoft’s maturation from Linux consumer to Linux distro vendor.

## LLM perspective
- View: This is Microsoft committing long-term to Linux, not just as a guest, but as a first-class platform asset.  
- Impact: Cloud customers gain a tightly integrated, audited, low-sprawl Linux that matches Azure’s tooling and compliance story.  
- Watch next: WSL integration quality, Fedora upstream contribution volume, and how aggressively Azure Linux is promoted over Ubuntu/RHEL images.
