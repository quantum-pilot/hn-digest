# Show HN: Eyeball

- Score: 229 | [HN](https://news.ycombinator.com/item?id=48367723) | Link: https://eyeball.rory.codes/

- TL;DR  
Eyeball is a minimalist browser game where you place a marker on a line to match a given number’s position as precisely as possible, optimized for mouse/trackpad accuracy. HN readers compare it to Matthias Wandel’s classic Eyeball, which tests broader geometric intuition and has clearer affordances. Some players found the new UI visually slick but initially confusing, then got hooked chasing near-perfect percentage errors. Suggestions include a repeat-training mode, mobile app support, and UI tweaks, while others note viral score-sharing driving new HN signups.
