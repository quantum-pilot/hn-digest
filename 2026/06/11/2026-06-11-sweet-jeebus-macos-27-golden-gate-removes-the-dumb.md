# Sweet Jeebus, macOS 27 Golden Gate Removes the Dumb Icons from Menu Items

- Score: 321 | [HN](https://news.ycombinator.com/item?id=48487435) | Link: https://daringfireball.net/2026/06/macos_27_golden_gate_removes_the_dumb_icons_from_menu_items

### TL;DR
macOS 27 “Golden Gate” removes the universally‑applied icons that cluttered every menu item in macOS 26 “Tahoe,” a change widely panned as ugly, inconsistent, and hostile to Mac conventions. Apple’s updated Human Interface Guidelines now explicitly say icons should be used “sparingly and with purpose,” only when they clearly communicate common or visual actions. Gruber reads this reversal as evidence that Apple’s design leadership has changed course, with the “rot” of fashion‑driven UI giving way to usability‑first thinking.
