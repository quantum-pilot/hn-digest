# The 29th International Obfuscated C Code Contest (IOCCC) 2025 Winners

- Score: 357 | [HN](https://news.ycombinator.com/item?id=48432199) | Link: https://www.ioccc.org/2025/

- TL;DR  
  The 29th International Obfuscated C Code Contest (2025) announces a large, high‑quality batch of winners after the 2020–2024 hiatus, helped by revamped rules, documentation and outreach. Judges introduce “fun challenges” attached to each winning entry, encouraging variants and explanations via GitHub PRs. Notable winners include a GameBoy emulator whose source code forms the console’s outline, a 366‑byte Subleq‑style Linux‑capable VM, multiple “hat‑trick” three‑time winners, and playful awards like “Most soothing” or “Ping pong prize.”
