# Amateur armed with ChatGPT solves an Erdős problem

- Score: 735 | [HN](https://news.ycombinator.com/item?id=47903126) | Link: https://www.scientificamerican.com/article/amateur-armed-with-chatgpt-vibe-maths-a-60-year-old-problem/

### TL;DR
A 23‑year‑old non–math-major, Liam Price, prompted GPT‑5.4 Pro with an unsolved Erdős problem about “primitive sets” (collections of integers where no number divides another). The model produced a long, messy proof sketch. Mathematicians Jared Lichtman and Terence Tao extracted a key new idea: using a known formula from a different area of number theory in a way no one had tried for this problem. They then turned that into a clean proof and already see broader applications to understanding large integers.
