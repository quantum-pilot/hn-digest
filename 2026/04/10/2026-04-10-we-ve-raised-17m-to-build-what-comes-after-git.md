# We've raised $17M to build what comes after Git

- Score: 303 | [HN](https://news.ycombinator.com/item?id=47712656) | Link: https://blog.gitbutler.com/series-a

- TL;DR  
  GitButler, founded by GitHub cofounder Scott Chacon, raised a $17M Series A led by a16z to build “what comes after Git”: collaboration infrastructure that keeps code, conversations, and AI-agent activity in sync. Their first product, an open-source CLI that layers on top of Git, focuses on stacked branches, multitasking, easier undo, and proactive conflict detection. Hacker News debates whether Git really needs redesigning, worries about VC-driven lock-in, and points to tools like Jujutsu as existing, lighter-weight evolutions.

- Comment pulse  
  Git isn't broken → core problems are UI, training, and workflow; Git plumbing already supports snapshots, branching, and custom porcelains like jj or magit.  
  VC skepticism → funding a GitHub cofounder signals 'next GitHub', raising lock-in and license fears — counterpoint: others shrug if the CLI remains OSS.  
  Strategic view → investors bet on 'AI-native' infrastructure centralizing agents, context, and conflicts; some welcome innovation but insist any Git successor be open and reliable.
