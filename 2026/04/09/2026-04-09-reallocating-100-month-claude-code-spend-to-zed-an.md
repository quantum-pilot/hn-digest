# Reallocating $100/Month Claude Code Spend to Zed and OpenRouter

- Score: 284 | [HN](https://news.ycombinator.com/item?id=47700972) | Link: https://braw.dev/blog/2026-04-06-reallocating-100-month-claude-spend/

### TL;DR
The author is frustrated that a $100/month Claude Code subscription still hits rate/usage limits, especially with bursty coding sessions. They switch to paying $10/month for Zed and auto‑adding $90/month to OpenRouter credits, which last up to a year and work across many models. Zed, Cursor, Claude Code, and CLI tools like OpenCode are treated as “agent harnesses” that can all be wired to OpenRouter, trading flat, time‑boxed subscriptions for per‑token, model‑agnostic usage that better fits irregular workloads.
