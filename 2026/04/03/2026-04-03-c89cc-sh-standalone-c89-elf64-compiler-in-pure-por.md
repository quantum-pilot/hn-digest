# C89cc.sh – standalone C89/ELF64 compiler in pure portable shell

- Score: 191 | [HN](https://news.ycombinator.com/item?id=47598413) | Link: https://gist.github.com/alganet/2b89c4368f8d23d033961d8a3deb5c19

### TL;DR
A single shell script, `c89cc.sh`, implements a full C89 compiler that outputs x86‑64 ELF binaries with a tiny built‑in libc. It runs in a “hermetic” mode (PATH set to empty) and relies only on portable shell features and its own parser/AST infrastructure, itself generated by a BNF‑to‑shell tool. Hacker News readers admire the technical stunt and bootstrapping implications, but question real‑world usefulness without tests, documentation, and a proper repository, and raise security concerns about shipping compilers everywhere.
