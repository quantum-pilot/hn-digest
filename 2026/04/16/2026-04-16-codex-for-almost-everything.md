# Codex for almost everything

- Score: 630 | [HN](https://news.ycombinator.com/item?id=47796469) | Link: https://openai.com/index/codex-for-almost-everything/

### TL;DR
OpenAI’s updated Codex turns its coding assistant into a general-purpose Mac co-pilot: it can see/click/type across apps, browse the web, generate images, work with 90+ plugins, and support end‑to‑end dev workflows (PR review, multi-terminals, SSH devboxes, rich file previews). New automations, long-lived tasks, memory, and context-aware suggestions let it resume work over days and proactively propose actions. Hacker News frames it as another step in the Claude/Codex convergence, debating genuine usability gains, security of OS-level control, and hype-heavy release tactics.
