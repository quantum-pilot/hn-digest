# Microsoft fire idTech team at Id software

- Score: 500 | [HN](https://news.ycombinator.com/item?id=48819244) | Link: https://gamefromscratch.com/microsoft-fire-idtech-team-at-id-software/

## TL;DR
Microsoft is reportedly laying off much of id Software’s idTech engine team, likely pushing future games toward Unreal Engine 5. Commenters frame this as cost-cutting and workforce “commoditization”: generic UE5 talent is cheaper and more replaceable than a bespoke engine team, but at the cost of technical distinctiveness and long-term innovation. Some see a strategic blunder that strengthens Epic’s engine monopoly; others note incomplete evidence on exactly who was cut and argue that freeing talent from Microsoft might ultimately help the wider industry.
