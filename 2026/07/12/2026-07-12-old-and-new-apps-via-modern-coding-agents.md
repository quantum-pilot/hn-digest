- TL;DR  
  Terry Tao is using modern coding agents/LLMs to build interactive supplements for his math work, sparking discussion on where these tools fit. Commenters report big gains using LLMs to rapidly create visualizations, classroom demos, and one-off research tools that were previously “nice-to-have” but unaffordable in time. Others stress that LLM-written code is often brittle and unmaintainable, so it’s best reserved for non‑mission‑critical, inspectable outputs—more like a microwave in a chef’s kitchen than a replacement for serious engineering.
