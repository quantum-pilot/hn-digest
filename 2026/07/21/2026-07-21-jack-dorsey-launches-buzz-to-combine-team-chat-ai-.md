# Jack Dorsey launches Buzz to combine team chat, AI agents and Git hosting

- Score: 213 | [HN](https://news.ycombinator.com/item?id=48995213) | Link: https://runtimewire.com/article/jack-dorsey-block-buzz-team-chat-ai-agents-git

### TL;DR
Buzz is Block’s new open-source, self-hostable workspace that merges team chat, a Git-based software forge, and AI agents around cryptographically signed Nostr events and a unified identity system. Humans and agents act as first-class participants with auditable actions, while each organization runs its own central relay instead of relying on Slack or GitHub. HN discussion splits between enthusiasm for protocol-controlled workspaces and concerns about agent safety, cloying UX, AI-built tool reliability, and Nostr/Git architectural bets.

### Comment pulse
- Shared agents complicate ACLs; some advocate single-user agents or treating agents as normal users — counterpoint: secret-scoping harnesses claim strong no-leak guarantees.  
- Several see the UI and marketing as hypey “slop”, distrust AI-built tools, and expect many such projects to be quickly abandoned.  
- Some welcome a Slack/Teams challenger but question Nostr’s suitability at large scale, tight Git coupling, and how agents, identities, and rich UI components will evolve.
