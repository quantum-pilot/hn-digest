# Decoding the obfuscated bash script on a Uniqlo t-shirt

### TL;DR
A blogger fully decoded the obfuscated Bash script printed on a Uniqlo x Akamai t-shirt, treating it as reverse-engineering puzzle art. Hacker News comments riff on the idea of returning shirts for syntax errors, debate how hard the code is to OCR given its quirky layout and base64 blob, and nitpick the typography (Roboto Mono with non-monospace-like kerning). Several people use it as an informal benchmark for modern OCR and multimodal LLMs, many of which handle it surprisingly well.
