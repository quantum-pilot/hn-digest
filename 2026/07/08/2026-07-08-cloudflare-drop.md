# Cloudflare Drop

- Score: 185 | [HN](https://news.ycombinator.com/item?id=48836233) | Link: https://www.cloudflare.com/drop/

### TL;DR
Cloudflare Drop is a frictionless way to host static sites: drag-and-drop a folder, get an instantly live, globally cached URL, then optionally “claim” it into a Cloudflare account. Links apparently auto-expire after about an hour if unclaimed, limiting persistence and abuse. HN notes it closely resembles Netlify Drop and earlier tools, but many still like having a Cloudflare-backed variant with CDN performance by default. Discussion centers on abuse potential versus existing guardrails, plus meta-arguing about HN’s increasing negativity.
