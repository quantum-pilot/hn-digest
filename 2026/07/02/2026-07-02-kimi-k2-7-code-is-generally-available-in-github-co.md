# Kimi K2.7 Code is generally available in GitHub Copilot

- Score: 396 | [HN](https://news.ycombinator.com/item?id=48756602) | Link: https://github.blog/changelog/2026-07-01-kimi-k2-7-is-now-available-in-github-copilot/

### TL;DR
GitHub Copilot now offers the Chinese Kimi K2.7 Code model, giving enterprises a non‑US option via a trusted Western provider at GPT‑5.4‑mini–like prices. Hacker News discussion, however, fixates less on Kimi itself and more on AI access models: painful Copilot price hikes, opaque auto model selection, and instability in cloud offerings are pushing many developers toward local/open models (Qwen, GLM, Gemma) and cheaper harnesses like Claude Code, Cursor, and custom setups.
