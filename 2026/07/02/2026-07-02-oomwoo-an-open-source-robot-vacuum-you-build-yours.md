# Oomwoo, an open-source robot vacuum you build yourself

- Score: 452 | [HN](https://news.ycombinator.com/item?id=48755005) | Link: https://makerspet.com/blog/building-an-open-source-robot-vacuum-meet-oomwoo/

### TL;DR
Oomwoo is an open‑source, build‑it‑yourself robot vacuum that emphasizes modular hardware, 3D‑printed parts, and community‑driven software. Commenters like the idea of a repairable, cloud‑free vacuum, but note piecemeal hardware is far more expensive than buying or modding existing consumer robots. The project’s GitHub/Discord, module‑based organization, and heavy use of AI “vibe coding” spark debate: some worry about practicality, while others see it as exactly how a single motivated person can bootstrap ambitious open hardware.
