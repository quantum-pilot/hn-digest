# ZCode: Claude Code from the Makers of GLM

- Score: 267 | [HN](https://news.ycombinator.com/item?id=48751752) | Link: https://zcode.z.ai/cn

### TL;DR
ZCode appears to be a Claude-Code-style AI coding assistant built by the team behind the Chinese GLM models. The Hacker News thread barely discusses capabilities; instead it focuses on the product’s web presence: the main site and docs default to Chinese, the English version is hard to discover on mobile, and the Linux beta flow jumps into Chinese-only Feishu. Commenters see this as a broader pattern of Chinese AI products struggling with internationalization and onboarding for non-Chinese users.
