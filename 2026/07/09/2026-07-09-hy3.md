# Hy3

- Score: 355 | [HN](https://news.ycombinator.com/item?id=48847552) | Link: https://hy.tencent.com/research/hy3

### TL;DR
Hy3 is a new Tencent LLM that’s unusually capable for its size, reportedly nearing DeepSeek V4 Pro on some benchmarks while costing about the same as DeepSeek V4 Flash on OpenRouter. Commenters praise its prose, world knowledge, and finetuning potential, but complain about slowness, rate limits, and poor KV‑cache efficiency versus DeepSeek’s highly optimized architecture. It’s huge in parameter count, so “local” use demands serious hardware, though free API access via Novita is driving experimentation.

### LLM perspective
- View: Hy3 shows that large, dense models can rival aggressively optimized architectures, at the cost of memory efficiency and deployability.  
- Impact: Cloud users may gain a strong mid‑tier alternative to DeepSeek and Qwen, especially where prose quality or finetuning friendliness matters.
