# Wordgard: In-browser rich-text editor from the creator of ProseMirror

- Score: 255 | [HN](https://news.ycombinator.com/item?id=48772573) | Link: https://wordgard.net/

- TL;DR  
Wordgard is a new in‑browser rich‑text editor from the creator of ProseMirror, rethinking the architecture rather than serving as ProseMirror v2. It still builds on contenteditable, but introduces design changes meant to fix long‑standing issues the author saw in ProseMirror, at the cost of no direct upgrade path. HN commenters want clearer documentation of those trade‑offs, praise the site’s standout illustrations, discuss schema/typing pain points, and lament that the web still lacks a standard, robust WYSIWYG model.
