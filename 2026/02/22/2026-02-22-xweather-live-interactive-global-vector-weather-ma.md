# Xweather Live – Interactive global vector weather map

- Score: 124 | [HN](https://news.ycombinator.com/item?id=47111626) | Link: https://live.xweather.com/

### TL;DR
Xweather Live is an ad‑free, browser-based global vector weather map built to showcase Xweather’s APIs. It offers smooth, real‑time timeline scrubbing and a large set of overlay layers in one interface, including temperature, wind, tropical cyclones, lightning, hail, renewable energy, and wildfire data. HN users praise its performance compared with tools like Windy, especially outside the US, and like the intuitive global temperature view. Suggestions include highlighting the hottest/coldest locations and exploring aviation- and climate-focused views.
