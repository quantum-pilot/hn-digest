# Tell HN: YC companies scrape GitHub activity, send spam emails to users

- Score: 582 | [HN](https://news.ycombinator.com/item?id=47163885) | Link: https://news.ycombinator.com/item?id=47163885

## TL;DR
Developers report YC-backed startups scraping GitHub commits and stars to harvest email addresses for unsolicited pitches, sometimes repeatedly. A GitHub employee says this clearly violates GitHub’s terms and accounts can be banned, but several users describe ignored or dismissed spam reports and weak enforcement. Commenters discuss the trade-off between open-source visibility and privacy, suggesting anonymized commit emails and avoiding starring repos. Most see this scraping as damaging, disrespectful developer marketing and blacklist companies that do it.
