# Termux

- Score: 319 | [HN](https://news.ycombinator.com/item?id=46854642) | Link: https://github.com/termux/termux-app

## TL;DR
Termux is an Android terminal emulator that ships a minimal Linux environment plus its own package repo, letting you install familiar CLI tools (ssh, clang, ffmpeg, rsync, etc.) and extend it via plugins (API, Boot, Widget, Styling, Tasker, Float). The project documents multiple install sources (F-Droid, GitHub, Play Store) with strict signature-compatibility rules and a security caveat for GitHub test-key builds, plus Android 12+ “phantom process” kill issues. HN users treat Termux as a core productivity and backup tool.
