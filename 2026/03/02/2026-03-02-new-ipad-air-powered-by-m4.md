# New iPad Air, powered by M4

- Score: 295 | [HN](https://news.ycombinator.com/item?id=47218175) | Link: https://www.apple.com/newsroom/2026/03/apple-introduces-the-new-ipad-air-powered-by-m4/

### TL;DR
Apple’s new iPad Air keeps the same $599/$799 starting prices but jumps to an M4 chip with 12GB unified memory, much faster GPU (ray tracing), a 3x-faster Neural Engine, and new N1/C1X chips for Wi‑Fi 7 and better cellular efficiency. iPadOS 26 finally adds a real windowing system, menu bar, improved Files, and a Preview app, pushing the iPad further toward laptop territory. HN discussion questions the lack of multi-user support, Apple’s deliberate iPad/Mac separation, and how many users need this power, while praising device longevity but worrying about software and server lock-in.
