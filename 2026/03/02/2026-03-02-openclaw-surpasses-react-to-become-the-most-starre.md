# OpenClaw surpasses React to become the most-starred software project on GitHub

- Score: 243 | [HN](https://news.ycombinator.com/item?id=47217812) | Link: https://www.star-history.com/blog/openclaw-surpasses-react-most-starred-software

### TL;DR
- OpenClaw, an AI agent/automation framework, has overtaken React as GitHub’s most‑starred non‑aggregator project, hitting 250k+ stars in under four months. HN readers question whether this reflects real impact, citing star inflation, bots, and AI hype versus React’s decade of proven influence. Some laud OpenClaw as a fun, natural‑language “universal glue” occasionally enabling impressive long‑running tasks, while others see mostly trivial automations, marketing spam, and a spotlight on how weak today’s underlying tooling and integrations still are.
