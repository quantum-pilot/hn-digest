# Vite 8.0 Is Out

- Score: 515 | [HN](https://news.ycombinator.com/item?id=47360730) | Link: https://vite.dev/blog/announcing-vite8

### TL;DR
Vite 8 replaces its historic dual-bundler setup (esbuild for dev, Rollup for prod) with Rolldown, a Rust-based bundler that keeps Rollup’s plugin API while delivering roughly 10–30x faster builds and major real-world reductions (minutes down to seconds). It’s now part of a unified toolchain with the Oxc compiler. New features include integrated devtools, tsconfig `paths` support, TypeScript `emitDecoratorMetadata`, WASM SSR, console log forwarding, and a lighter React plugin v6. Install size increases ~15 MB; migration is mostly drop-in with a compatibility layer and a recommended two-step path via rolldown-vite. HN comments highlight huge productivity gains, long-standing waste in build pipelines and testing setups, praise for Vite’s contributor experience, and some philosophical pushback on JS build complexity itself.
