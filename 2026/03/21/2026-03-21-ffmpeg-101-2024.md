# FFmpeg 101 (2024)

- Score: 207 | [HN](https://news.ycombinator.com/item?id=47463547) | Link: https://blogs.igalia.com/llepage/ffmpeg-101/

### TL;DR
The article is a hands-on “FFmpeg 101” for C developers: it briefly maps the FFmpeg command‑line tools and core libraries, then walks through building a minimal video player. It explains the main data structures (AVFormatContext, AVStream, AVCodec, AVCodecContext, AVPacket, AVFrame), shows how to demux a file, select streams, find the right decoder, and push packets through a FIFO-style decode loop to get raw frames. Build instructions use Meson/Ninja and auto-fetch FFmpeg. HN comments mainly share more in‑depth learning resources.
