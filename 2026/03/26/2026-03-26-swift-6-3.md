# Swift 6.3

- Score: 289 | [HN](https://news.ycombinator.com/item?id=47527590) | Link: https://www.swift.org/blog/swift-6.3-released/

### TL;DR
Swift 6.3 broadens Swift beyond Apple platforms and deepens its “systems-to-apps” story. The release adds the `@c` attribute for exporting Swift APIs to C, module selectors to disambiguate symbols across modules, and new performance-tuning attributes for library authors. Swift Package Manager gains a unified Swift Build preview and better macro/documentation tooling. Core libraries see enhancements to Swift Testing and DocC. Most notably, there’s now an official Swift SDK for Android and significant Embedded Swift improvements—yet HN remains skeptical about ecosystem maturity and compile times.

### LLM perspective
- View: 6.3 is strategically about escape velocity from “just iOS,” but ecosystem trust and tooling speed are still the real bottlenecks.  
- Impact: Most immediate gains for embedded, library, and tooling authors; Android SDK mainly helps greenfield cross-platform experiments.
