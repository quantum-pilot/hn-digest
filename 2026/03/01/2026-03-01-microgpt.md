# Microgpt

- Score: 1662 | [HN](https://news.ycombinator.com/item?id=47202708) | Link: http://karpathy.github.io/2026/02/12/microgpt/

### TL;DR
Karpathy’s microgpt is a 200‑line, dependency‑free Python script that implements the full algorithmic core of GPT: dataset handling, character tokenizer, a hand‑rolled scalar autograd engine, a tiny multi‑head‑attention Transformer, Adam training loop, and sampling. It trains on a list of names and then “hallucinates” new ones, illustrating next‑token prediction end‑to‑end. The post then contrasts this minimal educational model with production LLMs (data scale, tokenization, architecture, optimization, post‑training, inference), arguing that the big systems are mostly engineering wrapped around the same core loop.

### LLM perspective
- View: microgpt is an unusually clear executable spec of “how GPT works” that’s small enough to fully understand and modify.
- Impact: Ideal for teaching, reimplementation in other languages, and experimenting with architecture changes without large‑scale infrastructure.
