# Kagi Small Web

- Score: 683 | [HN](https://news.ycombinator.com/item?id=47410542) | Link: https://kagi.com/smallweb/

### TL;DR
- Kagi Small Web is an open-source, StumbleUpon-style viewer that surfaces recent posts from curated RSS-based personal blogs, comics and small YouTube channels, organized by topic and accessible via a “Next Post” button or feeds. It aims to highlight human-written, non-corporate sites inside Kagi search. HN commenters like the idea but argue its definition of “small web” is too narrow, biased toward frequently updated tech/AI blogs, and compare it with alternative human-curated blog directories and broader debates about Kagi’s core search quality.

### Comment pulse
- Name feels wrong → Index limits to recent RSS blogs/comics, biasing toward tech/LLM chatter and excluding static hobby sites and infrequently updated classics.  
- Desire for rarity-focused filters → Users want discovery of rarely updated sites instead of algorithms favoring frequent posters — counterpoint: recency helps keep browsing lively.  
- Parallel projects emerge → Commenters share curated blog lists and random-surf tools like minifeed, wiby, cloudhiker, echoing StumbleUpon-era serendipitous browsing.  
- Mixed views on Kagi search quality → Some see no improvement over Google’s noisy results; others praise cleaner answers, better small-site surfacing, and helpful summaries.
