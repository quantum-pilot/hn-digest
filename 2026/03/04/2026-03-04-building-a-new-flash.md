# Building a new Flash

- Score: 203 | [HN](https://news.ycombinator.com/item?id=47253177) | Link: https://bill.newgrounds.com/news/post/1607118

### TL;DR
Developer Bill Premo is building a modern, open-source replacement for Flash: a cross‑platform 2D animation and authoring environment written in C# (Avalonia + SkiaSharp). It closely recreates Flash’s workflow—timeline, vector merge drawing, symbol library, shape/motion tweens, camera, filters, rich text, audio, and SWF/HTML5 export—while adding .fla/XFL import and C# scripting with an ActionScript‑to‑C# transpiler. HN discussion is excited about backwards compatibility and the coder–artist workflow, but flags unproven claims and the Patreon‑before‑code optics.
