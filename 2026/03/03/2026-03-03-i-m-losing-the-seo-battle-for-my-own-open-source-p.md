# I'm losing the SEO battle for my own open source project

- Score: 424 | [HN](https://news.ycombinator.com/item?id=47232158) | Link: https://twitter.com/Gavriel_Cohen/status/2028821432759717930

- TL;DR  
    - An open-source project (NanoClaw) is being outranked in search by a quickly-registered fake domain that news sites mistakenly linked as its homepage. A seasoned SEO offers a recovery plan: fix backlinks from press, use Google Search Console, add schema/structured data, and build a consistent social + GitHub “entity” around the real site. Commenters are angry this busywork is needed at all, report weak performance from many search engines, and note that AI assistants sometimes identify the real project more reliably—but not consistently.
