# An update on Steam / GOG changes for OpenTTD

- Score: 253 | [HN](https://news.ycombinator.com/item?id=47442834) | Link: https://www.openttd.org/news/2026/03/19/steam-changes-update

- TL;DR  
  - OpenTTD’s team clarifies why Steam and GOG now require owning Atari’s Transport Tycoon Deluxe: it’s a negotiated compromise, not legal pressure. The game remains fully free on the project’s website; only platform distribution is conditioned. Atari will also help pay server costs, while OpenTTD keeps project independence. HN commenters mostly see this as an unusually constructive IP–fan collaboration, though some worry about platform power, opaque revenue terms, and what happens if Atari changes hands.

- Comment pulse  
  - Many praise the deal → Gating Steam/GOG behind TTD while staying free elsewhere respects Atari’s IP and preserves OpenTTD — counterpoint: Others see needless corporate appeasement.  
  - Platforms matter → Losing direct Steam/GOG access feels like banishment; users value one-click install, auto-updates, and discovery over manual downloads.  
  - Skeptics worry → Without transparency on revenue-sharing or legal terms, Atari’s contribution and goodwill could vanish if ownership changes.

- LLM perspective  
  - View: This is a rare negotiated middle path between copyright holders and a mature open-source clone, not a simple victory/defeat.  
  - Impact: Sets a template for fan projects on big stores: stay free off-platform, accept platform-specific conditions, negotiate infrastructure funding.  
  - Watch next: Whether Atari documents this arrangement, shares revenue, and maintains terms if Transport Tycoon’s rights or management change hands.
