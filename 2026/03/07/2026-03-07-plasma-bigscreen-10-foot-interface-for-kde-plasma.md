# Plasma Bigscreen – 10-foot interface for KDE plasma

- Score: 626 | [HN](https://news.ycombinator.com/item?id=47282736) | Link: https://plasma-bigscreen.org

- TL;DR  
  Plasma Bigscreen is KDE’s open‑source 10‑foot interface for TVs and set‑top boxes, essentially a couch‑friendly Plasma shell that runs standard Linux apps (Steam, Kodi, Jellyfin, browsers) on Wayland. It targets privacy‑respecting, non‑surveillance “smart TVs” with CEC, gamepad, keyboard and phone control plus big‑screen settings and customization. HN commenters like the idea but note the project is older, lightly resourced and not Kodi‑polished; debate whether a full desktop shell is overkill, and compare KDE’s power vs GNOME’s simpler UX.

- Comment pulse  
  - Project is long‑running, small‑team and not a core KDE focus → promising but rough; more flexible than Kodi because it’s a full desktop shell.  
  - Some want a minimal 10‑foot launcher on Openbox/RetroArch → big tiles, few widgets — counterpoint: full Plasma “complexity” enables real apps instead of kiosk‑like limits.  
  - KDE praised for aesthetics and customizability → others report bugs, overwhelming options and prefer GNOME’s opinionated, macOS‑like UX, especially for newcomers and lower‑maintenance desktops.

- LLM perspective  
  - View: Treat Bigscreen as an experimental shell for tinkerers, not a turnkey smart‑TV replacement yet.  
  - Impact: Best suited to privacy‑minded HTPC users already comfortable with Linux packaging, Flatpak, and occasional rough edges.  
  - Watch next: images, releases and performance reports on ARM TV boxes/handhelds; UX simplification could broaden appeal beyond existing KDE enthusiasts.
