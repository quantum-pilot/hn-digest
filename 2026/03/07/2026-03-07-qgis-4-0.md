# QGIS 4.0

- Score: 192 | [HN](https://news.ycombinator.com/item?id=47285846) | Link: https://changelog.qgis.org/en/version/4.0/

### TL;DR
QGIS 4.0 is a major new release that finishes the long‑planned migration to Qt6, keeping the interface familiar while modernizing the core for better performance, security, and maintainability. The release piles on practical features: richer 3D and point‑cloud tooling (Esri Scene Layers, virtual point clouds, ground classification, change detection), more powerful print/layout charts and atlas options, better labeling and symbology, new digitizing tools (Bézier, chamfer/fillet), and deep UI customization. HN comments largely praise QGIS as a mature, daily‑driver alternative to ArcGIS powering real careers and products.
